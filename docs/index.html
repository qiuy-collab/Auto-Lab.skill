Quick Start

把右边这段丢给 AI Agent，让它先初始化仓库和环境。

只做初始化

README 不重复写完整流程。真正的执行规则在 SKILL.md 里，让 Agent 去读就行。

请读取并初始化这个 skill：

https://github.com/qiuy-collab/Auto-Lab.skill

请只做初始化，不要开始生成报告。

需要完成：
1. 克隆仓库。
2. 阅读 SKILL.md。
3. 按 SKILL.md 检查运行环境。
4. 自动安装基础依赖。
5. 检查 vendor skills 是否完整。
6. 运行环境检查脚本。
7. 如果缺少 API Key、Word 模板、作业要求或其他文件，直接列出来让我补。
